# IGCreator — Next Iteration Plan

Based on comprehensive audit (March 2026). Score: 6.0/10 (up from 4.5).

1. **Wire backend image generation → frontend** — Studio generate button calls backend, shows real images in iPhone mockup
2. **Instagram OAuth + basic posting** — Complete OAuth flow, enable direct posting from Studio
3. **Stripe payment integration** — Checkout, subscription management, tier enforcement
4. **Replace fake testimonials** — Remove or replace with beta user quotes
5. **Privacy Policy + Terms of Service** — Legal pages

2. **Content repurposing** — Paste blog/transcript/notes → generate multiple IG posts
3. **Real analytics** — Connect Instagram Graph API for actual metrics
4. **Draft/history system** — Persist generated content in DB, show history panel in Studio
5. **Content idea generator** — Daily trending topic suggestions from strategy research

### Phase 3: Make It Exceptional (Growth)
1. **Reel storyboard editor** — Script breakdown, scene editing, audio, timing
2. **Carousel slide editor** — Multi-slide editing, drag-reorder, per-slide layout
3. **Hashtag research tool** — Volume, competition, trending indicators
4. **Story templates + scheduling** — Instagram Stories support
5. **Competitor analytics dashboard** — Real competitor benchmarking
6. **Calendar drag-drop + best-time suggestions**
